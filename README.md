# Virtstand

Under development...

## Development

* clone repo
* `npm install`
* either run `npm run build:link` and `virtstand`
* or `npm run build` and `node build/index`

## Requirements

* Virtualbox >=6.1.30
* Vagrant >=2.2.19
* Node.JS >=14.5.0

## Workarounds

For absolute paths MSYS (windows shell) changes them.
Use `MSYS_NO_PATHCONV=1` to avoid it.
